# Shadow Nexus 高审美升级版

已根据首页审美建议完成修改：

- 首页改为“沉浸式世界入口”布局
- 主视觉中心化：超大 Shadow + 紫月 + 雾气 + 粒子
- 左侧改为“暗影数据库 / 情报终端”
- 右侧改为“暗影战力终端”，增加头像、能量条和排行榜质感
- 世界设定保留为导航下方独立界面区域，包含世界观、时间线、地图、角色档案
- 规则下方增加 9 张规则图鉴 / 设定插画区域
- 排行榜扩展 100 人，支持垂直滚动
- 作品展示扩展为大型图床画廊
- 已内置 assets 初始图床 SVG，占位图可直接替换

## 使用方法

直接双击 `index.html` 打开即可。

替换图片时，保持 `assets/` 内文件名不变即可自动生效。
